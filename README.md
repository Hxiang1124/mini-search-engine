\# Mini Search Engine



A simple search engine built with Python and Flask.



\## Features



\- Inverted index for fast lookup

\- Multi-keyword search

\- Basic ranking (term frequency)

\- REST API support



\## Usage



Run:



python app.py





Then open:





http://127.0.0.1:5000/search?q=python
